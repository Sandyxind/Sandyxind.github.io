Command Summarize

  • Installing Hexo
    $ npm install hexo-cli -g
  • Creating Blog
    //Creating and initializing Blog. <folder> is the name of Blog
$ hexo init <folder>
  • Installing relevant packages
    $ cd <folder>
$ hexo install
  • Creating new pages
    $ hexo new page <folder>
  • Creating new post
    $ hexo new 'postname'
  • Testing Off-line
    $ hexo server
  • Testing Off-line
    $ hexo server
  • Transfering to Static website
    Using this command will create a public folder. Copy all files to GitHub Pages can update your website through GitHub Desktop App.
    $ hexo generate
